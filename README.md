# game_minesweeper

python3.5, pygame

Run game:
python3 main.py

install pygame:
https://www.pygame.org/wiki/GettingStarted

### About game
Win - open all cell no bomb
Game Over - open cell bomb

### Control:
- Left mouse button - open cell
- Right mouse button - flag on bomb, don`t work!!!!
- Key R - restart game
